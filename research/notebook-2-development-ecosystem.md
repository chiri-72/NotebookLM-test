## 핵심 인사이트

### 1. RAG 프레임워크 선택
- **LangChain**: 광범위한 LLM 오케스트레이션, multi-tool agents, chains
- **LlamaIndex**: 고품질 retrieval, 인덱싱 전략, RAG observability
- **병행 사용 가능**: LlamaIndex로 인덱싱 + LangChain으로 워크플로우 구성

### 2. MLOps 2026 트렌드
- **60% 빠른 모델 배포**, **40% 프로덕션 인시던트 감소**
- **90% 배포 시간 단축**, **95%+ 모델 uptime**, **40-60% 인프라 비용 절감**
- Feature stores가 기업 ML 확장성의 필수 요소로 자리잡음
- 거버넌스가 MLOps에 embedded, 사후 추가가 아님

### 3. Vector Database 비교
- **Pinecone**: 프로덕션 RAG 시스템에 최적, 자동 샤딩/복제, 비용 3-5배 높음
- **Weaviate**: 기능과 유연성의 균형, 하이브리드 검색 (dense vectors + BM25)
- **ChromaDB**: 프로토타이핑 및 소규모 배포, zero-config, embedded 아키텍처

### 4. MLOps 성숙도 지표
- 이터레이티브-점진적 개발
- 자동화 (통합, 테스트, 릴리스, 배포)
- 지속적인 모니터링 (모델 드리프트, 성능 저하 감지)
- 버저닝 및 재현성
- 자동화된 재학습 파이프라인
